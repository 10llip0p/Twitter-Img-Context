# Twitter-Img-Context
2017年6月現在,[twitter.com](https://twitter.com/)にて画像をモーダルで表示した際に隠されてしまうツイート本文を画像下に表示するChrome拡張.

## Usage
src.crxを[chrome://extensions](/chrome://extensions/)にドラッグアンドドロップ

## Note
margin: autoだと左に寄ってしまうので適当に20%にしてる.
